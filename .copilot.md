# Copilot Configuration

## Plugin

This plugin is a gateway for WooCommerce to allow merchants to use Alma's services for installment payments.
It uses two kinds of gateways:

- Alma Blocks gateway (default)
- Alma Checkout gateway (fallback if Blocks is not available)

I have 4 gateways:

- pnx
- paylater
- credit
- paynow

## instructions

### PHP

Code must be compatible with php 7.4 to 8.5

### JS

Never compile JS, I do it by myself with a watcher
